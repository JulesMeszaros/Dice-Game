# DICE DELUXE! (WIP)

Dice Deluxe est un projet de roguelike deck-builder inspiré des mécaniques du Yahtzee, développé sous le moteur LÖVE2D en LUA.
Son développement a débuté en mai 2025, et est aujourd'hui en cours.

Il s'agit du premier jeu développé par AEROSOL DELUXE Interactive.

## Installation et lancment du jeu

### A partir du code source

- Clone le repo github
- Installer Love2D sur la machine (et s'assurer que Love2D soit dans le PATH!)
- Aller dans l'invité de commande
- Se placer dans le dossier téléchargé
- Tapper `love ./`

### A partir du fichier .love
- Installer le moteur Love2D
- Telecharger la release souhaitée
- Double clicker le fichier pour l'ouvrir
- C'est parti!

Et c'est gagné!!!
