# DICE GAME (WIP)


LE jeu de Yatzee, mais en beaucoup mieux.

## Installation et lancment du jeu

### A partir du code source

- Clone le repo github
- Installer Love2D sur la machine (et s'assurer que Love2D soit dans le PATH!)
- Aller dans l'invité de commande
- Se placer dans le dossier téléchargé
- Tapper `love ./`

Et c'est gagné!!!
